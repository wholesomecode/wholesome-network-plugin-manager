Have you ever network disabled a plugin only to find it still active on one or more sub-sites? Use Network Plugin Manager to easily identify which sites your plugin is active on and quickly deactivate it.

== Installation ==

To install Network Plugin Manager:

1. Note that the plugin is only compatible with a WordPress Multisite Installation. 
2. Upload the plugin files to the `/wp-content/plugins/wholesome-network-plugin-manager` directory, or install the plugin through the WordPress network plugins screen directly.
3. Activate the plugin through the Network Admin \'Plugins\' screen in WordPress
